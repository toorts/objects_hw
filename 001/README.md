# Проблемы в проектировании

* В проектировании класса Animal следует тщательно оценить необходимость добавления методов toGo(), fly() и swim() и учитывать различия между животными в способах движения и других свойствах. 

* Если вышеперечисленные методы не являются универсальными для всех животных, то возможно более правильным подходом будет создание подклассов, которые учитывают специфические свойства и методы для каждого типа животных.
